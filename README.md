# Proyecto Power BI – Análisis de reciclaje

## Objetivo
Analizar hábitos de reciclaje, costes de recogida y perfiles de viviendas para identificar oportunidades de mejora y acciones de concienciación ciudadana.

## Herramientas utilizadas
- Power BI
- Power Query
- DAX
- Modelado de datos

## Análisis realizados
- Calidad de datos
- Costes de recogida
- Análisis temporal
- Segmentación de viviendas
- Comportamiento de reciclaje
